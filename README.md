# Student Performance Data Analysis

## Project Overview
This project performs data cleaning, exploratory data analysis (EDA), and visualization on a student performance dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data Cleaning
- Missing Value Handling
- Descriptive Statistics
- Outlier Detection
- Correlation Analysis
- Data Visualization

## Visualizations
- Histogram
- Box Plot
- Pair Plot
- Correlation Heatmap
- Bar Charts
